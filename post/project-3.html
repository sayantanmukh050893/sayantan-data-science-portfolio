Finding Messi

August 20, 2020

This project is a classic example of the Face Recognition capabilities of Computer Vision and Deep Learning with the help of which we would be creating a face recognizer for Lionel Messi.

In many big games critics find Messi to be subdued and missing from the game. This recognizer would end that debate once in for all as it would be able to identify Messi in images as well as in videos. Pun intended!!

Resources

  1. Facial images of Lionel Messi from Google Images (atleast 20 images) We can either accomplish it by the hard way by manually downloading all the 20 images or we can follow the smarter approach of downloading the images in a single batch all at once. In the upcoming days I would be writing a blog post on How to download a batch of images quickly from Google Images and would add the link here.

  2. The above downloaded images would be placed inside a folder named messi which is placed inside a folder named football_dataset

  3. encode_faces.py :- This is a python file which would help us to create 128-dimensional encodings for all the images that we have in our dataset. i.e. football_dataset in our case. This would output a pickle file named encodings.pickle. i.e. football_encodings.pickle in our case.

  4. examples :- This folder would contain the testing images with the help of which we would evaluate our recognizer.

  5. recognize_faces_image.py :- This python file helps us with the recognition of faces based on the input image provided from examples folder and encodings which were dumped as pickle file. The output of this file would be an image which would have faces tagged in them as either Unknown (if it is anyone else but Messi) or messi (if it is Messi)

  6. videos :- This folder would contain the testing videos with the help of which we would evaluate our recognizer.

  7. recognize_faces_video_file.py :- This python file helps us with the recognition of faces based on the input video provided from videos folder and encodings which were dumped as pickle file. The output of this file would be a video which would have faces tagged in them as either Unknown (if it is anyone else but Messi) or messi (if it is Messi)

  8. output :- This folder would contain the output videos after the recognizer has tagged the images with bounding boxes.
